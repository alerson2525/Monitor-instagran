Política de Privacidade

Última atualização: 22/08/2025

Este aplicativo (“App”) foi criado para integrar-se às plataformas da Meta (Instagram/Facebook) com a finalidade de exibir contagem de seguidores e informações básicas de perfil em um dispositivo externo (por exemplo, ESP8266/ESP32 + display).

Dados que Coletamos

  • Dados de perfil do Instagram autorizados pelo usuário via OAuth (por exemplo: ID do usuário, nome de usuário, foto de perfil e contagem de seguidores).
  • Métricas autorizadas pela API (quando aplicável), como número de seguidores. Não coletamos conteúdo de mensagens.

Como Usamos os Dados

  • Para exibir, em tempo real, a contagem de seguidores e dados básicos de perfil no dispositivo do usuário.
  • Para manter a sessão válida com a API da Meta (por exemplo, renovação de tokens).

Compartilhamento de Dados

Não vendemos, trocamos nem compartilhamos dados com terceiros, exceto quando exigido por lei ou necessário para cumprir requisitos das plataformas (Meta) ou de hospedagem.

Retenção e Exclusão

Armazenamos apenas o mínimo necessário (por exemplo, tokens de acesso) pelo tempo estritamente necessário para o funcionamento do App. Você pode solicitar a exclusão de dados entrando em contato pelo e-mail abaixo.

Segurança

Adotamos práticas razoáveis de segurança para proteger as informações processadas pelo App. Entretanto, nenhum método é 100% seguro.

Seus Direitos

Você pode revogar permissões a qualquer momento nas configurações da sua conta (Instagram/Facebook) e solicitar informações, correção ou exclusão dos seus dados.
